# mini-crm-backend
Mini CRM application that allows users to manage leads, contacts, notes, and reminders

### Endpoints
- User registration: `auth/register/`
- Login: `auth/token`
- Refresh token: `/auth/token/refresh/`
- Contacts: `api/contacts`
- Leads: `api/leads`
- Notes: `api/notes`
- Reminders: `api/reminders`
